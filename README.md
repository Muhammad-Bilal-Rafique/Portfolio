# Bilal.dev | Portfolio

The professional home for my work, projects, and development journey. This portfolio is built to showcase my transition from a curious learner to a performance-focused Full-Stack Developer.

## 🚀 Built With
- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS & Shadcn UI
- **Animations**: Framer Motion (for fluid, S-Rank user interactions)
- **Email**: Resend (for server-side contact form handling)
- **Deployment**: Vercel

## ✨ Key Features
- **Performance First**: Optimized for speed, reflecting my interest in Core Web Vitals.
- **Architectural Storytelling**: A dedicated "Proof of Work" timeline mapping my journey from V1 (Nexus) to V2 (Orbit).
- **Interactive UI**: Custom glassmorphism components and staggered entrance animations.
- **Always Available**: Fully responsive design with an integrated CV download and real-time internship availability status.

## 📈 The Journey
This portfolio is more than just a landing page; it's a reflection of my growth as a CS student at UVAS. It documents my learning path—from initial JavaScript concepts to modern AI integrations—and serves as a hub for my professional outreach.
